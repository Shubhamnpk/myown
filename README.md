# myown - Productivity Management App

A comprehensive productivity and task management application built with Next.js, designed to help users organize their work, track progress, and boost efficiency.

## üöÄ Features

### Core Functionality
- **Dashboard Overview**: Personalized dashboard with productivity metrics, streaks, and daily tips
- **Task Management**: Advanced todo list with priority levels (low, normal, high, urgent) and inline editing
- **Goal Tracking**: Set and monitor goals with progress bars, categories, due dates, and sub-tasks
- **Notes System**: Organize and manage personal notes
- **Productivity Analytics**: Track daily productivity scores, view trends with interactive charts, and maintain streaks
- **Focus Timer**: Pomodoro-style timer to maintain concentration during work sessions
- **Resource Library**: Store and organize links to useful resources
- **Music Player**: Integrated music player for background ambiance
- **Study Session Scheduler**: Plan and schedule focused study periods
- **Calendar Integration**: Visual calendar widget for date-based planning

### User Experience
- **Guest Mode**: Try the app without registration (data stored locally)
- **User Accounts**: Secure registration and login system
- **Customizable Dashboard**: Rearrange and hide/show widgets to suit your workflow
- **Resizable Popups**: Open multiple modules simultaneously in floating windows
- **Dark/Light Themes**: Switch between themes for comfortable viewing
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Real-time Clock**: Display current time with timezone support

### Technical Features
- **Local Data Storage**: All data persists locally using browser storage
- **Modern UI**: Built with Radix UI components and Tailwind CSS
- **TypeScript**: Fully typed for better development experience
- **Animations**: Smooth transitions and micro-interactions with Framer Motion

## üì¶ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/myown.git
   cd myown
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   pnpm install
   # or
   yarn install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   pnpm dev
   # or
   yarn dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)
